@startuml

actor Administrativo
participant Sistema

activate Administrativo
Administrativo -> Sistema: 1 inicia definicao de nova categoria de tarefa
activate Sistema
Sistema --> Administrativo: 2 mostra a lista de areas de atividade e solicita\na descricao da categario de tarefa e a\nescolha de uma area de atividade
deactivate Sistema
Administrativo -> Sistema: 3 introduz a descricao e seleciona a area de atividade pretendida
activate Sistema
Sistema --> Administrativo:4 mostra a lista de competencias tecnicas associadas a area\n de atividade especificada e solicita a escolha de uma delas
deactivate Sistema
group loop
    Administrativo -> Sistema: 5 seleciona competencia tecnica solicitada
    activate Sistema
    Sistema --> Administrativo:6 mostra graus de proficiencia relativos a competencia tecnica selecionada
    deactivate Sistema
    Administrativo -> Sistema: 7 seleciona o grau de proficiciencia minimo
    activate Sistema
    Sistema --> Administrativo:8 solicita indicacao do seu caracter (i.e. obrigatoria ou desejavel)
    deactivate Sistema
    Administrativo -> Sistema: 9 introduz informacao pedida
    activate Sistema
end
Sistema --> Administrativo:10 apresenta dados e solicita confirmacao
deactivate Sistema
Administrativo -> Sistema: 11 confirma
activate Sistema
Sistema --> Administrativo:12 informa operacao bem sucedida
deactivate Sistema
deactivate Administrativo

@enduml
